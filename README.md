# Centrar Elementos

En los archivos **CSS** van encontrar diferentes clases para asignarle al elemento **#element**. Por defecto este último tiene la clase **"center"**.

## Lista de Clases
* center
* top-left
* top-medium
* top-right
* left-medium
* left-bottom
* bottom-medium
* right-medium
